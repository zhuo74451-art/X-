# CoinMeta X Hot Follow Engine v0.1 模板库
 
账号定位：
CoinMeta / 币界网中文 X 账号，不是传统新闻号，不是公告搬运号，而是中文 Web3 热点型准 KOL 官号。
 
目标：
把热点、快讯、大号原帖、链上数据、事件日历，改写成适合中文 X 传播的内容。
 
第一版只启用 5 个模板：
 
---
 
## 1. news_card_take
中文名：快讯卡片 + 一句话点评
 
适用：
- 币界网自家快讯
- 已有新闻卡片 / 快讯截图
- 有热点潜力，但原始快讯太像新闻稿
- 能提炼出 Web3 / AI / 市场角度
 
结构：
主帖：
1. 一句钩子
2. 一句真正的判断
3. 配图使用币界网快讯卡片
 
首评：
补充 2-3 个背景点或后续观察点。
 
常用开头：
- 这条消息的重点，不在标题里。
- 这事有点意思。
- 如果只把它当成普通快讯，可能会漏掉关键点。
- 真正值得看的不是这个事件本身，而是它背后的市场变化。
 
示例：
这条消息的重点，不在标题里。
 
真正值得看的不是某项目又发了一个更新，而是它背后代表的链上基础设施竞争正在继续加速。
 
首评：
补三点背景：
 
1. 这类更新直接影响用户体验和链上吞吐。
2. 如果多个项目同时加速升级，说明基础设施竞争可能重新升温。
3. 后续可以继续看开发者活跃度、交易量和生态项目迁移情况。

## 2. authority_quote_plus
中文名：权威大号转述 + 中文补充
 
适用：
- Lookonchain
- Arkham
- Whale Alert
- Spot On Chain
- 项目官方
- 交易所官方
- 头部 KOL 的数据帖
- 有明确事实锚点的大号原帖
 
结构：
主帖：
1. 说明权威来源发现了什么
2. 用中文讲清楚事件
3. 提炼一个 CoinMeta 角度
 
首评：
补充背景、风险、观察指标。
 
常用开头：
- Lookonchain 披露了一起典型案例。
- Arkham 这条数据值得看。
- 这不是单纯的链上异动。
- 这类数据最值得看的不是金额，而是行为变化。
 
示例：
Lookonchain 披露了一起典型的 FOMO 交易案例。
 
一名交易者在 Solana 链上高位买入 Meme 资产，随后相关叙事快速消失，最终出现大额亏损。
 
真正值得注意的不是这一次亏了多少钱，而是 Meme 交易里，叙事消失的速度有时比价格下跌更快。
 
首评：
这类案例可以看三点：
 
1. Meme 交易高度依赖叙事和流动性同步存在。
2. 一旦传播源消失，价格支撑会迅速变弱。
3. 对普通用户来说，这更像风险样本，而不是交易机会。

## 3. quick_data_take
中文名：数据快讯 + 一句话判断
 
适用：
- BTC / ETH / SOL 数据异动
- ETF 流入流出
- 巨鲸异动
- 链上地址变化
- 稳定币供应
- 板块回撤
- 清算数据
- Token Unlock
 
结构：
主帖：
1. 一个明确数据点
2. 一句解释这个数据为什么重要
3. 一句克制判断
 
首评：
列出后续观察指标。
 
常用开头：
- 这个数据不能只看表面。
- 今天最值得看的不是涨跌，而是这组数据。
- 这不是简单的价格波动。
- 这个信号更像是资金行为变化。
 
示例：
AI Agent 板块今天集体回撤。
 
真正值得看的不是单个代币跌了多少，而是资金是否正在从高波动叙事里撤出。
 
如果这种回撤继续扩散，后续需要观察交易量、KOL 讨论热度和链上资金是否同步降温。
 
首评：
可以重点看三件事：
 
1. 交易量是缩小还是放大。
2. 大额钱包是否开始减仓。
3. KOL 讨论是继续扩散，还是开始转向风险提醒。

## 4. event_calendar_watch
中文名：本周 / 下周 Web3 重点事件
 
适用：
- 下周几件大事
- 本周 Web3 市场事件
- 宏观日历
- Token Unlock
- 项目升级
- ETF / 监管时间点
- 重大会议 / 财报 / 数据公布
 
结构：
主帖：
1. 本周 / 下周有几件事值得关注
2. 列出 3-5 件事件
3. 每件事说明可能影响什么
4. 结尾提醒：市场交易的是预期，不只是结果
 
常用开头：
- 下周加密市场有几件事值得关注。
- 本周 Web3 市场不缺事件。
- 如果只看价格，很容易错过真正的时间点。
- 市场很多时候交易的不是结果，而是预期。
 
示例：
下周加密市场有几件事值得关注：
 
1. 美国宏观数据公布，可能影响 BTC 风险资产定价。
2. 某头部项目进入解锁窗口，市场需要观察潜在卖压。
3. ETF 资金流入是否延续，会影响主流资产情绪。
4. AI Crypto 相关项目进入关键更新周期，可能带动叙事回温。
 
真正值得看的不是这些事件是否“利好”，而是市场是否已经提前交易了预期。
 
首评：
看这类事件，不要只看当天结果。
 
更重要的是：
 
1. 事件前有没有提前拉升。
2. 事件后有没有兑现回落。
3. 市场有没有把注意力转向下一条主线。

## 5. hot_explainer
中文名：热点解释型短帖
 
适用：
- KOL 热议
- AI / Web3 叙事
- 跨市场热点迁移
- 用户知道这个词，但不知道为什么重要
- 需要把复杂热点讲成人话
 
结构：
主帖：
1. 为什么大家突然讨论这个？
2. 三点解释
3. 一句总结
 
常用开头：
- 为什么市场突然开始讨论这个？
- 这波热度不是突然来的。
- 表面上看是一个热点，背后其实是另一条线。
- 如果只看表层，很容易看错这件事。
 
示例：
为什么市场突然开始讨论 AI Agent？
 
核心原因有三个：
 
1. 链上自动化交易重新升温。
2. 多个 KOL 开始讨论 Agent 入口。
3. 部分项目开始从概念走向产品。
 
真正值得关注的不是 AI Agent 这个词还火不火，而是它有没有进入第二轮叙事重定价。
 
首评：
这条线后续可以看三个确认信号：
 
1. 有没有真实用户使用。
2. 有没有持续收入或手续费。
3. 有没有从概念炒作转向产品交付。

## 视觉 Prompt 规则
 
visual_prompt_cn 只输出图片提示词，不生成图片。
 
三种视觉模式：
 
1. coinmeta_card_reuse
用于 news_card_take。
说明：复用币界网快讯卡片，必要时加标题包装。
 
2. data_terminal_style
用于 quick_data_take / authority_quote_plus。
说明：深色背景、链上数据、K线、资金流向、真实截图优先。
 
3. research_infographic_style
用于 event_calendar_watch / hot_explainer。
说明：时间线、表格、分层结构、研究报告长图。
 
禁止：
- 不要让 Image2 编造具体数字。
- 数据图必须基于真实数据。
- 不要生成真实人物肖像。
- 不要做夸张投资承诺图。
- 不要生成容易被误解成官方公告的图。
